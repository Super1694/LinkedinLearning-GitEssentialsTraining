
# GIT: Undoing Changes (Restore)

You can restore from a number of levels. First Remember the levels
 #### Top down
- GIT Online Repo
- Git Local Repo
- Git Local Staging Area
- Git Local File

Moving Up we do The Following

1. save file (ctrl + s, unless autosave is on)
1. git add . (adds changes to staging area)
1. git commit -m "message" (Move staged snapshot into local repo history)
1. git push (Upload commits to github)

## Status command (git status)

Hugely powerful and helpful command. This will give you suggestions on how to walk-back changes (mostly making these notes unnecessary - but they are a good learning practice)
